Working on Simulations

This page is intended to get started or continue working on analytics and simulations. Beginning with the login (or registartion for new users), we'll take you through the various steps os setting up your project.

Getting started means that you begin by selecting the type of application that you're working on, This includes a wide rnage, such as healthcare, manufacturing, finance, etc. After you select the project type, you'll be asked to enter the system and graphical paremeters. This inmformation will be used to configure your project, so that we can proceed with the analytics and simulations.

If you're continuing a project that you've started previously, then the previous configuration will appear automatically when you select your project. Once you've configures your project, you'll be ready to display the results. The more detailed your configuration, the more complete the results will be. Hence it's likely that you'll want to periodically view the interim results as you gradually configure the system.

To view the results as reports, select the 'Display Reports' item from the menu at the top of the page.
