How to Display Simulations

This page is the starting point for choosing which type of simulations to display. While the results of the system's Analytics can be visualized via the 'Display Analytics' page, the various tabs of the 'Display Simulations' page are intended specifically to show the various types of Simulations available to you.

For example you can select the types of forecacts and scenerios thata re used to create the Simulations. similarly, you can choose between 2D and 3D animations, as well as how to configre them. For more detailed explanationsof the various types of Simulations available to you, please see the system's Help screens.

To adjust the system or graphics parameters of the Simulations (e.g. lighting, texture, etc), return to the 'Working on Simulations' item from the menu at the top of the page.
